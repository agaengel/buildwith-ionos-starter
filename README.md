# Buildwith IONOS Starter

## Features

* Gridsome
* Tailwindcss
* SCSS
* Markdown
* Tags
* Blog

## Install

### 1. Install Gridsome CLI tool if you don't have

`yarn install --global @gridsome/cli --frozen-lockfile`

### 2. Create project

`yarn deploy`
